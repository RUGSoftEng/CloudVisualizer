What is Cloud Computing?

Cloud computing is a service which allows you to store and access data, programs and other resources over the internet instead of on the local storage of your device. This is useful if your device or network is not adequate to support the resources you need. There are multiple cloud providers who offer this service of leasing resources.

What is the usage of this site?

This web application allows you to compare prices between different cloud providers based on settings that can be adjusted. It is possible to choose between Google Cloud, Amazon Web Services and Microsoft Azure as the current provider. You can change settings regarding the Database, Storage and Virtual Machine options you want to have in your selection. After having configured settings in each category, you can drag your selection to the canvas, after which you can calculate a current price or go into advanced settings to specify further details. Multiple calculations can be compared in a graph, where each line represents a specific calculation. The advanced settings also show a graph which shows the immediate difference in price when you change an option there, allowing you to make a decision on which settings are preferable.
